## 개요
- 사용자가 이력서를 작성하고 수정할 수 있으며, 자신의 포트폴리오를 손쉽게 게시할 수 있다.
- 자신의 경력과 역량을 효과적으로 돋보일수 있도록 돕는 것이 목표
- Spring-boot 및 security 학습
